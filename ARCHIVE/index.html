Scripting

For over a decade, Python has been my go-to scripting language. I first picked it up using Blender, using it as part of a modeling and animation workflow. That early dive into creative coding gave me both a strong technical base and a fascination with making art through code. From there, I explored Processing, unlocking ways to add motion and interactivity to my generative graphics. That journey eventually brought me to p5.js, where I could bring those same concepts into the browser. Along the way, I expanded into JavaScript and modern web development — blending code, design, and interaction into a single creative toolkit.

Automation

While coding began as a personal passion, it quickly became an essential part of my professional toolkit. As a Linux Administrator at Blue Sky Studios, I used Python and Bash to automate a wide range of daily workflows, improving efficiency and reducing manual overhead. I also worked extensively with orchestration and configuration management tools such as Ansible and Puppet to scale these automations across the studio’s infrastructure. In recent years, I’ve expanded my automation skill set to include Salt, Kubernetes, Helm, Rancher, and Terraform — enabling me to manage complex, distributed environments with the same efficiency and reliability I first achieved through scripting.

AWS Cloud

Systems Engineering & Cloud Infrastructure

Scripting taught me an important lesson early on — you don’t always need to build everything from scratch. Leveraging existing tools and platforms can unlock tremendous potential. That mindset, combined with the rise of cloud computing, led me to explore AWS. Over three years, I experimented with AWS for personal projects, including a 2021 build of a Blender-integrated render farm service hosted entirely in AWS. This project deepened my understanding of distributed computing and further fueled my interest in cloud solutions. After several months of focused study, I earned my AWS Solutions Architect Associate certification, solidifying my expertise in designing scalable, reliable cloud architectures.
